# Sessionless C/C++ Library

## Dependencies 
* [secp256k1](https://github.com/bitcoin-core/secp256k1)

    > _**NOTE:**_ when using CMake the dependencies are downloaded automatically

## Evironment setup 
CMake and C++ compiler are required.

## Build the demo
```shell
mkdir build
cmake -B ./build -S .
cd build
make
```
